# The Learning Resources

## A Vue app using the CLI and components to practice all the vue concepts that I learned so far.

## Some Used Concepts:
  * Components
  * Teleport
  * provide/inject
  * component :is
  * keep-alive
  * props
  * emits

### How to run:
  * Have npm & Vue CLI installed
  * Open a terminal in project folder
  * npm install
  * npm run serve
